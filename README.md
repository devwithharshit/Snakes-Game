# Snakes-Game
 A classic Nokia-style Snake game built with Pygame, featuring high scores, smooth controls, and background music. 🎮🐍

How to Play Nokia Snake Game
1. Start the Game
Run the Python script to start the game.
The welcome screen will appear with the following options:
Press Space Bar to Start the Game
Press Backspace to Exit

2. Control the Snake
Use the Arrow Keys to move the snake:
Right Arrow → Move Right
Left Arrow → Move Left
Up Arrow → Move Up
Down Arrow → Move Down

3. Eat the Food
A red square represents food. Move the snake to eat the food and score points.
Each food increases your score by 10 points.
The snake grows longer after eating food.

4. Avoid Collisions
If the snake collides with the wall or itself, the game will be over.

5. Game Over & Restart
When the game is over, the screen will display:
"Game Over! Press Enter To Restart"
"Press Backspace to Quit!"
Press Enter to restart the game.
Press Backspace to exit the game.

6. High Score
The game saves your highest score in hiscore.txt.
Try to beat your own high score!

7. Sound Effects
Background music starts when the game begins.
A game-over sound plays when you lose.
